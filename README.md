# Homebrew Tap for Jenie

This is the official Homebrew tap for [jenie](https://github.com/Nattothemoon/jenie-cli), a CLI tool for interacting with Xano backend APIs.

## Installation

```bash
brew tap Nattothemoon/jenie
brew install jenie
```

## Usage

```bash
jenie              # Launch TUI mode
jenie help         # Show all commands
jenie login        # Authenticate
```

## Updating

```bash
brew update
brew upgrade jenie
```

## Uninstall

```bash
brew uninstall jenie
brew untap Nattothemoon/jenie
```
